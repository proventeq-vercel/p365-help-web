# Proventeq365 — User Guide

**Proventeq365** is a tool for monitoring the Microsoft 365 environment, particularly SharePoint Online and OneDrive. It provides information about redundant, obsolete, and trivial (ROT) content, overshared content, and permissions that may be set incorrectly. The product assists users and administrators in identifying and addressing risks related to sharing with unintended users, including external parties. Content Monitor regularly notifies users and administrators of such issues and suggests actions and policies for remediation. Its use can support enhanced security and organisation within the Microsoft 365 environment.
