# Notification

When click on Notification Menu, it will open following screen

![Notification illustration 1](images/image105.png)

Notifications will appear on this screen based on settings. Hovering over a notification displays its Delete icon, allowing users to remove notifications individually. A Delete icon near the Notification header lets users delete all notifications at once.

![Notification illustration 2](images/image106.png)
