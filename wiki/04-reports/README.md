# Reports

For Reports, there are two types available: Permission and ROT Analysis. Additionally, a summary tab presents an overview of both permissions and ROT data obtained from the source.

**Summary** -- This summary provides an overview of both Permissions and ROT data collected from the source.

**Permissions** -- This report presents detailed information regarding permissions that have been overshared across documents and folders.

**ROT** -- This report offers a comprehensive analysis of ROT (Redundant, Obsolete, and Trivial) data within the source. Documents are categorized as Duplicates, Obsolete, or Trivial. Users can access ROT analysis as a consolidated view or as separate reports for Duplicate, Obsolete, and Trivial documents.
