# Exchange Monitoring

The **Exchange Monitoring** feature provides visibility into Exchange Online mailboxes, storage usage, and activity patterns within your organisation, specifically focused on **links** and **attachments**.

The Links & Attachments view in Exchange Monitoring provides visibility into how links and file attachments are shared across your organisation. It helps identify potential risks related to data sharing, external exposure, and large file distribution.

## Key Capabilities

- **Attachment Analysis** — Monitor emails containing file attachments, including file types, sizes, and volume, to identify heavy usage or unnecessary data sharing.
- **Link Sharing Visibility** — Track emails that include **shared links** (e.g. SharePoint or OneDrive links) to understand how content is being distributed.
- **External Sharing Insights** — Identify links and attachments sent to **external recipients**, which may pose security or compliance risks.

## How Exchange Monitoring Works

1. Create a workspace with the Exchange Monitoring type and configure it as needed.
2. Once data analysis is complete, use the Email and Link reports to get insight into how links and attachments are distributed across locations and departments.
